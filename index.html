    FoxxiBot is an all-purpose Twitch & Discord Bot

    Through FoxxiBot, the aim was to create a bot that's easy to use but also extendable via Javascript-powered .NET plugins via JINT (a .NET Javascript Interpreter) allowing new features to be coded and added on the fly.

    Created to be Simple to Use

    The aim was to make sure that anyone could use FoxxiBot, out of the box it will have ways to add commands for both Twitch and Discord easily.

    Extensive for Bigger Streams

    For those who need more, there's ways to write your own plugins and add to the bot's web panel for more specialized features you may need.

    Plugins to Add More Experiences

    Through the Plugins system, you'll be able to extend the bot via scripts written my the creator(s) and community. The sky's the limit!
