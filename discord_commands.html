Discord Commands

Command Alias Response Scope
!avatar [argument] The bot will return the specified users avatar image Everyone
!prefix Returns the current bot's prefix Everyone
!prefix set [argument] Sets the bots prefix if you wish to change it Administrator
!promo Returns the status of the Promo System Everyone
!promo add [argument] Will add a users Twitch stream to the system if given a valid username Everyone
!promo remove [argument] Will remove a users Twitch stream from the system if given a valid username Everyone
!purge [argument] The bot will attempt to delete the number of messages specified (messages older than 2 weeks cannot be deleted) Administrator
!say [argument] The bot will say whatever you want Everyone
!userinfo [optional argument] The bot will send back information on the current user or a specified user Everyone
!status [type] [link] [text] Sets the bots status to whatever you'd like! (for more info, join the Discord) Administrator
!system serverid Registers the Discord Server ID with the bot if not done on first run (needed for web panel integration) Administrator
!system syncchannels Will re-sync all channels into the database if they're not up-to-date (if the bot is running when changes are made, you won't need to do this) Administrator
!system syncroles Will re-sync all roles into the database if they're not up-to-date (if the bot is running when changes are made, you won't need to do this) Administrator
!system botchannel Sets the channel you're currently in as the Bot Channel (not recommended if using plugins) Administrator
!system botchannel [on/off] Will activate or de-activate the Bot Channel limitation Administrator
!system greetingchannel Sets the channel you're currently in as the Greeting Channel Administrator
!system greetingchannel [on/off] Will activate or de-activate the Bot Greeting System Administrator
!system promochannel Sets the channel you're currently in as the Stream Promo Channel Administrator
!system promochannel [on/off] Will activate or de-activate the Stream Promo Notifications System Administrator
!system autorole [role] Sets the channel you're currently in as the Stream Promo Channel Administrator
!system autorole [on/off] Will activate or de-activate the Auto-Role System Administrator
